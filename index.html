Profile

Saya adalah mahasiswi Fakultas Ilmu Komputer, Jurusan Informatika di Universitas Klabat. Saat ini, saya akan memasuki semester 5 dengan Indeks Prestasi Kumulatif (IPK) terakhir sebesar 3.61. Selama menjalani studi di Universitas Klabat, saya telah mengembangkan keterampilan di bidang Web dengan mempelajari HTML dan CSS. Saya akan selalu bersemangat untuk mempelajari teknologi baru dan mencari solusi kreatif untuk masalah yang kompleks. Saya berharap dapat berkontribusi lebih dalam di bidang teknologi informasi dan terus mengembangkan diri saya melalui berbagai pengalaman praktis dan profesional.

Project

Project pertama yang saya buat adalah sebuah website portofolio, yang bisa anda cek disini. Website ini adalah final project yang saya kerjakan, dibuat hanya dengan menggunakan HTML dan CSS. Website ini berisi deskripsi personal tentang kelompok saya. Saya berharap website ini dapat memberikan gambaran yang jelas tentang kemampuan dan pengalaman dalam pengembangan web.
